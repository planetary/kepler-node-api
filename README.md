# Kepler-NodeJS-API

Screenshot service (nodejs API)

## Install

```bash
npm install --save kepler-api
```

## Usage

```javascript
kepler = require('kepler-api')

kepler.configure({
    'apiUrl': 'http://your-kepler-server',
    'projectSlug': 'your-project-slug',
    'apiKey': 'your-api-key'
    'build': 'build-number-or-blank-for-new-build'
})
```
